# somnia_bot

## 数据

```js
privite_keys.txt // 私钥，一行一个
proxies.txt // 代理，一行一个
```

## 执行

```
npm install
node start
```

## 功能 

- $STT 领水
- mint $PING $PONG
- autoswap $PING $PONG
- 定时执行
- 多线程执行

基于 [https://github.com/ziqing888/Somnia-bot](https://github.com/ziqing888/Somnia-bot) 、[https://github.com/vVANare/somnia-testnet-bot/](https://github.com/vVANare/somnia-testnet-bot/) 缝合
